# Preparation for Unity e-Portfolio (Setup Required)

## Overview

Before the Unity e-Portfolio presentation, all participants should install the required software so we can start directly with the practical demo during the session.

During the presentation we will:
- create a small 3D scene,
- work with GameObjects and physics,
- import assets,
- and write simple C# scripts in Unity.

---

# 1. Install Unity Hub & Unity Editor

Official download page:

https://unity.com/download

## Required Steps

- Download and install **Unity Hub**
- Install a Unity version  
  *(recommended: latest LTS version)*

During installation, make sure to include:

- Microsoft Visual Studio (Windows/macOS) or another C# IDE
- Platform support modules if available

---

# 2. Unity Account

You will most likely need a free Unity account to use Unity Hub and the Unity Editor.

## Notes

- You can create the account during installation or beforehand
- Unity Personal is free for students and personal use

---

# 3. Operating System Notes

## Windows
- Fully supported
- Recommended for the easiest setup experience

## macOS
- Fully supported

## Linux
- Unity Hub is available
- Setup may require additional configuration

---

# 4. Estimated Setup Time

The installation and setup process may take around:

- 20–30 minutes
- depending on internet speed and hardware

Please complete the installation before the presentation.

---

# 5. Optional Preparation

If you want to prepare in advance:

## Create a new Unity Project

1. Open Unity Hub
2. Click **"New Project"**
3. Select the **3D Template**
4. Create a project with any name

This step is optional because we will go through the setup together during the session.

---

# 6. Troubleshooting

If you encounter any issues during setup, feel free to contact me before the presentation.

Especially Linux users should test the installation earlier so potential problems can be solved in advance.
